# LIKELION 2023 community communication 서비스 (COCO) 
> 멋쟁이사자처럼 교내 아이디어톤 2등 수상작 🏆

COCO 는 내가 속한 커뮤니티 내에서 의사소통을 할 수 있도록 하는 서비스입니다. 이 서비스를 통해 이용자들은 자신이 살고있는 물리적 공간의 이웃들과 가상의 공간에서 소통을 통해 유대관계를 만들고 공동의 목표를 이룰 수 있습니다.
